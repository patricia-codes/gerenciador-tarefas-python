## 📌 Sobre o Projeto

Este projeto consiste em um **Gerenciador de Tarefas desenvolvido em Python puro**, criado com o objetivo de praticar **conceitos fundamentais da linguagem**, como estruturas de controle, manipulação de dados e leitura/escrita de arquivos.

A aplicação funciona diretamente no **terminal**, permitindo ao usuário gerenciar tarefas de forma simples.

---

## ⚙️ Funcionalidades

O sistema permite:

- ➕ **Adicionar tarefas**
- 📋 **Listar tarefas cadastradas**
- ❌ **Remover tarefas**
- 💾 **Salvar tarefas em arquivo JSON**
- 📂 **Carregar tarefas previamente salvas**

---

## 🧠 Conceitos de Programação Utilizados

Durante o desenvolvimento foram aplicados os seguintes conceitos de Python:

- Tipos de dados
- Estruturas de controle (`while`, `if`)
- Funções
- Listas e dicionários
- Manipulação de arquivos
- Serialização de dados com **JSON**

---

## 💻 Exemplo de Uso

Ao executar o programa, o seguinte menu é exibido no terminal:

** Menu de Opções **

1 - Adicionar tarefa

2 - Listar tarefas

3 - Remover tarefa

4 - Salvar tarefas

5 - Sair


O usuário pode selecionar a opção desejada para gerenciar suas tarefas.
